# network-model
Network model of two phase flow in porous media

## Packages required:
### Eigen
For solving sparse linear equations
##
    sudo apt install libeigen3-dev
    
### Cairo
Graphics library, figure out how to install this, there is nothing like this. Trust me.
##
    sudo apt install libcairo2-dev
    
### nlohmann-json
The output data is in json, which will make reading it for processing much easier than output.txt
##
     sudo apt install nlohmann-json3-dev
